# BTC-to-USDT-for-Bitbar

Shows the last Bitcoin price (in USD) on the Poloniex exchange for [Bitbar](https://getbitbar.com/)

### Installation:

Just click on this [link](https://tinyurl.com/ycv58zne) and plugin automatically set.
